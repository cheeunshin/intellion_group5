
## Team Principles

<div align="center">

| Principle | Meaning |
|:--:|:--|
| **Clarity** | 자료와 코드를 누구나 보기 쉽게 정리 |
| **Consistency** | 파일 구조와 문서 형식을 통일 |
| **Responsibility** | 맡은 역할을 끝까지 수행 |

## Notes

- 프로젝트 관련 자료와 코드는 지속적으로 업데이트될 예정입니다.
- 저장소 구조는 프로젝트 진행 상황에 따라 변경될 수 있습니다.
- 문서, 코드, 결과물은 팀 협업 기준에 맞춰 관리합니다.
